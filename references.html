References

All assets and code not credited here are my own work. Where third-party material has been incorporated, the licence and how it was used are noted. The three 3D models (character, apple, headphones) were modelled by me in Blender - only textures and materials were sourced externally.

Libraries

  • Three.js (r160)
    https://threejs.org/ - MIT licence
    Core WebGL renderer, scene graph, geometry, materials, lighting. Add-ons used: GLTFLoader for 3D model loading, EffectComposer + UnrealBloomPass for the wallpaper post-processing chain, OrbitControls for the Models app camera.

Textures & Materials

  • Apple base colour texture
    https://www.texturecan.com/details/185/
    Free PBR texture pack from texturecan, used as the base colour and roughness for the green apple model. Modified in Blender via a Cycles node graph (Color Ramp to shift hue toward green, plus a custom paint slot for the handwritten "blue-cat@japan.com" inscription) and baked to flat PBR textures for glTF compatibility.
  • Headphone materials (cup metal, cable, grille cloth)
    https://www.blenderkit.com/ - free assets, royalty-free use
    Material textures only (brushed aluminium, matte black plastic, speaker grille cloth) were sourced from BlenderKit and applied to the headphone model I built from scratch in Blender. The geometry, UV unwrap, and final PBR setup are my own work.
  • Lily Chou-Chou CD artwork (cover, rear insert and disc label)
    Cover Art Archive via MusicBrainz (official Lily Chou-Chou release)
    I pulled the artwork for this from MusicBrainz, which is where the official cover scans for the release are kept. Three images went onto the model: the front sleeve, the rear insert, and the printed disc label. I built the CD myself in Blender and laid flat (planar) UVs over the sleeve faces and the disc so each scan sits on the geometry without stretching.

Visual Inspiration

  • All About Lily Chou-Chou (リリイ・シュシュのすべて, 2001)
    Film directed by Shunji Iwai
    Visual influence for the wallpaper scene (green grass fields, soft cinematic palette), the character's contemplative pose, the green apple as a recurring symbol, and the Sony MDR-style headphones worn by the protagonist. No copyrighted footage or imagery is reproduced in the final submission.
  • macOS interface design (Apple Inc.)
    Apple macOS Big Sur / Sonoma visual language
    UI inspiration - dock, traffic-light window controls, menu bar, Notes-style document layout, Safari toolbar and tab bar. All UI elements are built from scratch in HTML/CSS; no Apple assets are bundled.
  • Canvas LMS (University of Sussex)
    Sussex Canvas instance interface
    The page chrome for this submission/sitemap/references trio is a visual homage to the university's Canvas LMS. Layout, colour palette (#013035), and typography are reproduced in HTML/CSS; no Canvas code or assets are bundled. The University of Sussex logo is used as a visual reference.
  • Blender (5.x) interface
    https://www.blender.org/ - GNU GPL
    Visual reference for the Models app chrome (workspace tabs, viewport toolbar, outliner, properties panel). All colours pixel-sampled from real Blender screenshots and reproduced in HTML/CSS - no Blender assets are bundled.

Tutorials & Learning Resources

The following YouTube tutorials were referenced while learning the Blender workflows used in this project. I'd never seriously used Blender before this module, and these were essential for getting the character model rigged and the headphones modelled.

  • "Create a 3D Low Poly Character (even if you've never opened Blender)" - Crashsune Academy
    https://www.youtube.com/watch?v=EQHdi5rX5fs
    Used as the foundational tutorial for the character model. Followed the modelling, UV unwrapping, rigging, and weight painting sections.
  • "Creating a 3D character in Blender - Turnaround & Modelling" - sorry stacy
    https://www.youtube.com/watch?v=qO3D7LuV3Ys
    Reference for the character's turnaround setup and the head/torso modelling workflow.
  • "Lowpoly 3D Hair - Blender Tutorial" - sorry stacy
    https://www.youtube.com/watch?v=kGSChMmW6x4
    Used for the character's hair modelling and weight painting approach.
  • "Creating a 3D character in Blender - UV Unwrapping & Texturing" - sorry stacy
    https://www.youtube.com/watch?v=JfuImB0_MrQ
    Followed for the UV unwrapping and final texturing pass on the character.
  • "Blender 3D Tutorial: Step-by-Step Airpods MAX Headphones" - 4RAYS TECH
    https://www.youtube.com/watch?v=jqn-e1kg8zc
    Reference for the headphone modelling workflow. The end result was adapted to a Sony MDR-G72 form factor rather than the Airpods Max shape.

Three.js Examples

Several Three.js official examples were referenced for the implementation patterns used in this project.

  • GLTFLoader example
    https://threejs.org/examples/?q=gltf#webgl_loader_gltf
    Reference for the async glTF loading pattern used in both the wallpaper scene and the Models app.
  • Post-processing - UnrealBloomPass
    https://threejs.org/examples/?q=bloom#webgl_postprocessing_unreal_bloom
    Reference for the EffectComposer + UnrealBloomPass chain used on the wallpaper scene for the dreamlike highlight bloom.
  • OrbitControls
    https://threejs.org/examples/?q=orbit#misc_controls_orbit
    Camera control reference for the Models app's Blender-style free-orbit camera.
  • ViewHelper / gizmo
    https://threejs.org/examples/?q=helper#webgl_helpers
    Reference for the XYZ axis gizmo in the Models app viewport corner. The final implementation is custom (the built-in ViewHelper had a positioning issue inside our window layout).

    SVG paths matched to the Sussex Canvas LMS interface for the decorative rail in submission / sitemap / references pages. Used as visual reference only.
  • System fonts only
    -apple-system, BlinkMacSystemFont, "SF Pro Text"
    No web fonts loaded. The site falls back through the system stack so it renders in SF Pro on Apple platforms, Segoe UI on Windows, and the default sans-serif elsewhere.
